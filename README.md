# Comment Purifier | 评论净化器

A lightweight Chrome extension to filter toxic comments and spam on any website.

轻量级浏览器插件，过滤网页中的恶意评论和广告。

---

## Features | 功能特性

### Filter Modes | 过滤模式

- **Hide Comments** - Completely hide comments containing bad words  
  **隐藏评论** - 完全隐藏包含敏感词的评论
  
- **Replace with Spaces** - Replace bad words with blank spaces  
  **替换为空格** - 将敏感词替换为空格
  
- **Replace with \*\*\*** - Mask bad words with asterisks  
  **替换为星号** - 用 *** 遮盖敏感词
  
- **Blur Text** - Blur the content, hover to reveal  
  **模糊文字** - 模糊显示，鼠标悬停可查看
  
- **Highlight** - Mark with red border for review  
  **高亮标记** - 用红框标记可疑内容

### Other Features | 其他功能

- **Whitelist** - Disable filtering for specific websites  
  **白名单** - 为特定网站停用过滤
  
- **Custom Keywords** - Add your own filter words  
  **自定义关键词** - 添加自己的屏蔽词
  
- **Built-in Keywords** - Includes common toxic words in English and Chinese  
  **内置词库** - 包含常见的中英文敏感词

## Installation | 安装方法

### English

1. Download this repository (Code → Download ZIP)
2. Unzip the folder
3. Open Chrome → `chrome://extensions/`
4. Enable **Developer mode** (top right)
5. Click **Load unpacked** → Select the unzipped folder

### 中文

1. 下载本仓库（Code → Download ZIP）
2. 解压文件夹
3. 打开 Chrome → `chrome://extensions/`
4. 开启**开发者模式**（右上角）
5. 点击**加载已解压的扩展程序** → 选择解压后的文件夹

---

## Usage | 使用说明

### English

1. Click the extension icon in toolbar
2. Toggle ON/OFF to enable/disable
3. Click the whitelist button to skip current site
4. Choose your preferred filter mode
5. Add custom keywords (comma separated)
6. Click Save

### 中文

1. 点击工具栏中的插件图标
2. 切换开关来启用/停用
3. 点击白名单按钮可跳过当前网站
4. 选择你喜欢的过滤模式
5. 添加自定义关键词（逗号分隔）
6. 点击保存
