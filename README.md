# Comment Purifier (评论净化器) 🛡️

**Protect your inner peace from online toxicity.**
(拒绝网络暴力，守护内心平静。)

> 🌍 **Comment Purifier** is a lightweight, privacy-focused Chrome Extension designed to filter out malicious comments, toxic content, spoilers, and spam on any website.
>
> 🇨🇳 **评论净化器** 是一款轻量级、注重隐私的浏览器扩展，专为净化网络环境设计。它可以帮你一键过滤掉网页上的垃圾评论、恶意辱骂、剧透内容以及烦人的广告。

---

## ✨ Features (核心功能)

* **💥 Super Strong Mode (超强力净化)**
    * Completely hides the entire comment block or post containing malicious keywords.
    * 独家“连坐”算法：发现脏话，直接隐藏整条评论或帖子。
    * *Optimized for (完美适配): YouTube, X (Twitter), Bilibili, Weibo, Reddit, etc.*

* **🔥 Strong Mode - Smart Spacing (强力净化 - 空格模式)**
    * Replaces toxic words with blank spaces to maintain the original layout.
    * 将敏感词替换为空格，保持段落排版整洁，不影响阅读体验。

* **🌫️ Blur Mode (模糊模式)**
    * Blurs sensitive content (Anti-spoiler). Hover to reveal.
    * 将内容高斯模糊，鼠标悬停时才显示。非常适合防止剧透。

* **👀 Highlight Mode (高亮模式)**
    * Marks suspicious keywords with a red box for easy review.
    * 仅用红框标出敏感词，适合内容审查或家长监管。

* **🌍 Universal Support (全网通用)**
    * Works on almost all modern websites with comment sections.
    * 内置智能容器识别算法，支持几乎所有网站的评论区过滤。

## 🚀 Installation (安装指南)

1.  **Download** the code from this repository (Click `Code` -> `Download ZIP`).
    (下载本仓库代码。)
2.  Open Chrome and go to `chrome://extensions/`.
    (打开 Chrome 浏览器，进入扩展程序页面。)
3.  Toggle **Developer mode** (右上角打开“开发者模式”).
4.  Click **Load unpacked** and select the folder.
    (点击“加载已解压的扩展程序”，选择你下载并解压的文件夹。)

## 🛠️ Tech Stack (技术栈)

* **Manifest V3**: Compliant with the latest Chrome extension standards (符合最新谷歌插件标准).
* **MutationObserver API**: For real-time filtering on dynamic websites (用于监听动态网页变化，支持瀑布流).
* **Shadow DOM & Regular Expressions**: Advanced text matching and element hiding (高级正则匹配与 DOM 操作).
* **i18n**: Built-in support for English and Chinese (内置中英双语支持).
